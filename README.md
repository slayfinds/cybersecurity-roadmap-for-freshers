# Cybersecurity Roadmap for Freshers and Career Switchers

A no-nonsense, free-first roadmap for moving into cybersecurity from any background.

> Goal: help freshers and people switching domains go from "I don't know where to start" to "I have a plan, a portfolio, and I'm applying" - without spending lakhs on courses you don't need yet.

## Why this exists

A lot of people who want to move into security get stuck because:
- Every "course" online promises a shortcut
- There's no clear order of what to learn first
- Nobody tells you what's actually free vs. what's a paid certification you need *later*, not now

This repo is an opinionated, step-by-step plan to fix that.

## How to use this repo

1. Not sure which path fits you? Start with [`PATHS.md`](./PATHS.md) - a no-hype comparison of SOC, VAPT, Network Security, and GRC
2. Pick your roadmap:
   - [`ROADMAP-SOC.md`](./ROADMAP-SOC.md) - SOC / Blue Team
   - [`ROADMAP-VAPT.md`](./ROADMAP-VAPT.md) - VAPT / Penetration Testing
   - [`ROADMAP-NETWORK-SECURITY.md`](./ROADMAP-NETWORK-SECURITY.md) - Network Security
   - [`ROADMAP-GRC.md`](./ROADMAP-GRC.md) - GRC / Compliance
3. Use [`RESOURCES.md`](./RESOURCES.md) - curated free resources, labs, and communities (shared across paths)
4. Track your progress with the checklists (just edit the `[ ]` to `[x]`)
5. Build the portfolio items as you go - that's what gets you interviews, not just certificates

## Who this is for

- Freshers and students figuring out where to start in security
- Anyone switching domains into cybersecurity, regardless of what field you're coming from
- Anyone who's been told "do a course" but has no idea which one

## Quick orientation: pick a lane

Cybersecurity is broad - SOC, VAPT, Network Security, and GRC are genuinely different jobs that just share the word "security." See [`PATHS.md`](./PATHS.md) for a full comparison, but the short version:

| Path | Good fit if you... | Typical entry role |
|---|---|---|
| **SOC / Blue Team** | Like monitoring, investigating, structured work | SOC Analyst L1 |
| **VAPT / Red Team** | Like breaking things, CTFs, offensive mindset | Junior Pentester |
| **Network Security** | Like infra, networking, building things securely | Network Security Engineer (Jr) |
| **GRC / Compliance** | Like policy, audits, process, communication | GRC Analyst |

Whatever your background, think honestly about which skills you already have that transfer (problem-solving, writing, infra knowledge, attention to detail) and pick the path that plays to them.
