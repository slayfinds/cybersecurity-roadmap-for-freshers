# Resources

All free unless marked 💰. Curated, not exhaustive - better to use 5 of these well than bookmark 50.

## Learning platforms

- **TryHackMe** - "SOC Level 1" path is built specifically for this transition. Free tier covers a lot.
- **Cybrary** - free intro courses on security fundamentals
- **Professor Messer** - free, excellent, full Security+ video course on YouTube
- **NetworkChuck / John Hammond (YouTube)** - practical, beginner-friendly security and networking content

## Practice labs

- **Splunk Free** (Splunk Free tier, single-user) - build a basic SIEM lab (SOC)
- **ELK Stack (Elastic, Logstash, Kibana)** - free, open-source SIEM alternative (SOC)
- **LetsDefend.io** - SOC-analyst-specific simulated environment, free tier available (SOC)
- **Blue Team Labs Online** - blue-team-focused practice challenges, free tier (SOC)
- **OverTheWire (Bandit)** - classic beginner Linux/security wargame, completely free (all paths)
- **HackTheBox** - CTF-style boxes, free tier covers a lot (VAPT)
- **DVWA / bWAPP / OWASP Juice Shop** - deliberately vulnerable web apps for legal practice (VAPT)
- **GNS3** - free network simulator for routing/switching/firewall labs (Network Security)
- **pfSense / OPNsense** - free firewall software, install on a VM for hands-on practice (Network Security)
- **Cisco Packet Tracer** - free, beginner-friendly network simulation tool (Network Security)

## Reference & frameworks

- **MITRE ATT&CK** (attack.mitre.org) - the standard reference for attacker tactics/techniques
- **NIST Cybersecurity Framework** - nist.gov, free PDF
- **OWASP** (owasp.org) - especially OWASP Top 10 if you later touch app security

## Certification study material (free study, exam itself is paid)

- CompTIA Security+ - Professor Messer's free course covers the full syllabus
- Blue Team Level 1 (BTL1) - has free sample modules and a strong reputation in SOC hiring circles

## Communities (India-relevant + global)

- r/cybersecurity, r/AskNetsec (Reddit)
- Local DEFCON groups / null (India's open security community, has chapters in many cities - good for in-person networking)
- LinkedIn cybersecurity groups - useful for India-specific job postings and referrals

## Job search

- Search titles: "SOC Analyst", "Security Analyst L1", "Cyber Security Analyst Fresher", "Junior Penetration Tester", "GRC Analyst"
- Naukri, LinkedIn, Instahyre for India-specific listings
- Don't ignore internal moves if you're already employed in a related field - many companies prefer promoting internal candidates with demonstrated security interest into junior security roles

---

💡 **Tip:** Pick 2–3 resources max at a time. Bouncing between ten different platforms is the #1 way people stall out before making real progress.
